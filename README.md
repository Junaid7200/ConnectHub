# ConnectHub

React Native (Expo Router) app with Supabase backend and Redux state.

## Run
```bash
npm install
npx expo start
```

## Tech
- Expo Router + React Native
- Supabase (auth, DB, storage)
- Redux

## Notes
- Types live in `src/types/types.tsx`.
- Supabase client in `src/lib/supabase.ts`.
- See `connectHub_context.md` for detailed project context/status.
